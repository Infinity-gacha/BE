# 💿 페르소나톡(BE) : DISC 기반 AI와의 대화 시뮬레이션 프로그램
> **2025 4-1 캡스톤 프로젝트**

페르소나톡은 다양한 성격 유형(DISC 기반)의 페르소나와의 대화를 통해 타인의 관점을 이해하고, 자신의 감정과 경계를 탐색하는 공감 훈련 프로그램입니다. AI 기반의 시뮬레이션을 통해 참가자는 주도적으로 대화를 이어가며 각기 다른 성향을 지닌 인물들과 소통하는 경험을 할 수 있습니다.
